# Haskell

## Instructions
1. To run, make sure you have ghc (The Glasgow Haskell Compiler) installed https://www.haskell.org/ghc/
2. Compile the executable with ```ghc --make hello.hs``` in the haskell directory.
3. Run the executable with ```./hello```
